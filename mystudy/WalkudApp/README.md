##Stetho
Facebook开源工具Stetho让Chrome调试Android非常方便，无需Root使用查看数据库、SharedPreferences等数据

###基本功能
1. 网络抓包(Http)
2. 数据库查询(允许使用[SQL](http://www.w3school.com.cn/sql/sql_intro.asp)操作)
3. SharedPreferences查询、修改
4. Android 视图树属性查看(API >=15)
